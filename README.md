# 五子棋游戏

这是一个使用Python和Tkinter库实现的五子棋游戏，支持多种游戏模式。

## 功能特性

- **单人游戏**：对战电脑AI（简单AI）
- **双人游戏**：本地对战
- **网络对战**：支持主机和客户端两种角色

## 项目结构

```
gobang-python/
├── main.py                # 游戏主入口
├── game_mode_selector.py  # 游戏模式选择器
├── gobang_base.py         # 五子棋游戏基类
├── gobang_single.py       # 单人游戏模式
├── gobang_multiplayer.py  # 双人游戏模式
├── gobang_network.py      # 网络对战模式
├── requirements.txt       # 项目依赖
└── README.md              # 项目说明
```

## 安装与运行

### 环境要求

- Python 3.6+
- tkinter（Python标准库，无需单独安装）

### 运行方法

1. 克隆或下载项目到本地
2. 进入项目目录
3. 运行以下命令启动游戏：

```bash
python main.py
```

## 游戏操作

1. **选择游戏模式**：启动游戏后，在主界面选择游戏模式
2. **落子**：在棋盘上点击鼠标左键进行落子
3. **重置游戏**：点击"重置游戏"按钮重新开始游戏
4. **返回主菜单**：点击"返回主菜单"按钮返回游戏模式选择界面

## 网络对战说明

1. **作为主机**：选择"双人游戏（网络对战）"，然后选择"创建游戏"，等待客户端连接
2. **作为客户端**：选择"双人游戏（网络对战）"，然后选择"加入游戏"，输入主机IP地址进行连接

## 游戏规则

- 黑白双方交替落子
- 先在一条直线（横、竖、斜）上连成五颗棋子者获胜
- 棋盘大小为15x15

## 技术实现

- 使用Python的tkinter库实现图形界面
- 使用面向对象编程思想，通过继承复用代码
- 网络对战使用socket库实现
